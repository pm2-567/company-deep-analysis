## 0. 行业适配选择器(速查)

> **红线规则**：业务模式标签 → 护城河框架是**六选一互斥**关系(消费品牌/平台网络/制造/金融/医药/地产)。选了哪一框架,**全篇护城河分析只用这一套**,**禁止**在主框架之外再开"经典 6 类护城河补充""现代扩展 4 类补充"等子小节。即使公司有混合业务(如腾讯有游戏+平台、美团有外卖+酒旅),也按**主业占比最高的业务模式标签**选择唯一框架,不用双框架并行。

> **完整选择器 + 触发规则**已在 `SKILL.md` Step3.3.0 给出,本节只列速查表。

| 业务模式 | 选用框架 | 权威背书 |
|----------|---------|---------|
| 消费品 / 品牌 / 零售 | §1 消费品牌护城河 | Aaker《Building Strong Brands》+ Keller《Strategic Brand Management》 |
| 互联网 / 平台 / 电商 / 社交 / 内容 / 游戏 / SaaS / 工具 | §2 平台与网络效应护城河 | Helmer《7 Powers》+ Thompson《Aggregation Theory》 |
| 制造 / 重资产 / 材料 / 能源 / 汽车零部件 | §3 制造业护城河 | Porter《Competitive Advantage》+ Clark & Fujimoto《Product Development Performance》 |
| 金融 / 银行 / 保险 / 券商 / 资管 | §4 金融业护城河 | Damodaran《Investment Valuation》+ 巴菲特 1995 致股东信 |
| 医药 / 生物科技 / 医疗器械 | §5 医药护城河 | Christensen《The Innovator's Prescription》+ 专利组合经典 |
| 地产 / 建筑 | §6 地产护城河 | Porter 五力 + 拿地能力 / 融资成本 |

## 1. 框架 A:消费品牌护城河

### 核心评估维度

| 护城河类型 | 关键问题 | 量化指标 | 权威背书 |
|----------|---------|---------|---------|
| **品牌资产** | 品牌能否持续产生溢价? | 价格溢价幅度、毛利率 vs 同行、复购率、NPS 净推荐值 | Aaker(1996) 品牌资产 10 指标模型 |
| **渠道控制** | 是否掌握终端渠道? | 直营门店数、加盟门店数、自有渠道占比、坪效 | Aaker(2000) 品牌组合战略 |
| **用户心智占位** | 提到该品类消费者第一个想到谁? | 品类第一提及率(TOM)、品牌搜索指数 | Keller(2008) CBBE 模型 |
| **产品创新力** | 能否持续推出爆款? | 新品收入占比、新品 3 年留存率 | BCG《The Innovation Game》(2007) |
| **规模效应** | 采购 / 生产 / 广告规模摊薄成本 | 销售费用率、采购成本 vs 同行 | Porter(1985) 规模经济 |
| **文化护城河**(新增) | 品牌是否绑定特定生活方式 / 文化? | 社交媒体话题量、跨界合作、KOL 主动推荐 | Holt《How Brands Become Icons》(2004) |

### 反问机制(消费品牌)

通用三轮反问见 §8。**消费品牌侧重点**:
- 可复制性:新品牌从 0 到 TOM 品类第一需要多少年?
- 周期韧性:经济下行 20% 时,品牌溢价能否维持?(例:茅台 vs 普通白酒)
- 隐蔽威胁:有没有新一代消费者正在用新品牌替代你?(Lululemon vs 传统运动品牌)

## 2. 框架 B:平台与网络效应护城河(互联网)

> 这是**最复杂的框架**,因为互联网公司的护城河来源最多元。

### 核心评估维度(Hamilton Helmer 7 Powers 中的 5 种 + 现代扩展)

| 护城河类型 | 关键问题 | 量化指标 | 权威背书 |
|----------|---------|---------|---------|
| **网络经济**(核心) | 用户/节点越多,价值是否越大? | Metcalfe 定律 V = n² ;跨边网络效应强度 | Helmer(1998); Reed's Law |
| **规模经济** | 单位成本是否随规模下降? | CAC / 营收(应随规模下降) | Helmer(1998) |
| **转换成本** | 用户切换到竞品的成本? | 数据迁移成本、关系链重建成本、API 锁定 | Helmer(1998) |
| **品牌**(平台品牌) | 平台品牌是否值得信任? | 用户主动搜索量、App Store 评分、用户投诉率 | Helmer(1998) |
| **数据资产**(现代扩展) | 数据飞轮是否已经转起来? | 数据量、年增量、数据多样性、模型效果 | Thompson(2017); Agrawal et al.《Prediction Machines》(2018) |
| **生态绑定**(现代扩展) | 第三方应用/商家是否愿意留在你的平台? | 第三方开发者数量、API 调用量、平台 GMV 增速 | Iansiti & Lakhani《Competing in the Age of AI》(2020) |
| **聚合优势**(独家) | 平台是否已"赢家通吃"或"寡头化"? | CR3 份额(>70% 算赢家通吃) | Thompson(2014) Aggregation Theory |

### 互联网护城河三轮反问(升级版)

> **方法论**:Andrew Chen《The Cold Start Problem》(2015)指出,网络效应有"启动-成熟-衰退"三阶段,
> 反问必须**结合阶段**才有意义。
> 通用三轮反问(可复制性/周期韧性/隐蔽威胁)见 §8,**本节只列互联网特有内容**。

**互联网特有第一轮:护城河阶段判断**

```
该护城河处于哪个阶段?
├── 启动期(网络效应尚未形成)
│   └── 反问:用户增长是否触发了网络效应?N 增长后 V 是否显著提升?
├── 成熟期(护城河已建立,稳定变现)
│   └── 反问:边际增长是否仍带来边际价值?
└── 衰退期(新平台冲击)
    └── 反问:用户是否开始多归属/迁移?
```

**互联网隐蔽威胁特别项**(在 §8 隐蔽威胁基础上扩展):
- **新模式替代**:Web3 替代 Web2?短视频替代图文?
- **监管威胁**:反垄断、数据安全、未成年人保护
- **跨平台颠覆**:Apple/Google 通过系统级能力直接绕过平台
- **技术替代**:大模型对搜索/推荐/客服的颠覆

## 3. 框架 C:制造业护城河

### 核心评估维度

| 护城河类型 | 关键问题 | 量化指标 | 权威背书 |
|----------|---------|---------|---------|
| **规模经济** | 采购/生产/销售的规模效应 | 产能利用率、单位成本随规模下降斜率 | Porter(1985) |
| **工艺专长** | Know-how 是否需要长期积累? | 良品率、单位能耗、专利数量 | Hayes & Wheelwright《Restoring Our Competitive Edge》(1984) |
| **客户认证壁垒** | 客户切换是否需要重新认证? | 客户认证周期(汽车 2-3 年、医疗 3-5 年)、认证产品收入占比 | Helper & Sako《Supplier Relations in Japan》(1995) |
| **产品组合宽度** | 能否一站式满足客户? | SKU 数量、交叉销售率 | Porter(1985) 产品差异 |
| **地理布局** | 产能是否靠近客户? | 海外基地数、运输半径 | 供应链管理经典 |
| **供应链控制**(现代) | 原材料/关键零部件是否自主? | 垂直一体化率、关键原材料自给率 | Christopher《Logistics & Supply Chain Management》(2016) |
| **专利组合**(现代) | 专利组合是否构成有效壁垒? | 发明专利数、核心专利被引用次数、PCT 国际专利 | Granstrand《The Economics and Management of Intellectual Property**(1999) |

### 反问机制(制造业)

通用三轮反问见 §8。**制造业侧重点**:
- 可复制性:新进入者要复制规模需要多少资本?客户认证需要多少年?(汽车 2-3 年、医疗 3-5 年)
- 周期韧性:行业下行 20% 时,公司毛利率压缩多少?(用上一轮行业下行期数据,例:周期股 20% 下行期毛利率压缩 2-3pct)
- 隐蔽威胁:海外反倾销/关税、新材料替代、加工费持续下行压力

## 4. 框架 D:金融业护城河

### 核心评估维度

| 护城河类型 | 关键问题 | 量化指标 | 权威背书 |
|----------|---------|---------|---------|
| **牌照壁垒** | 是否持有稀缺牌照? | 牌照种类(银行/保险/券商/资管/支付)、监管评级 | 中国银保监会/证监会牌照体系 |
| **资本规模** | 资本金是否构成壁垒? | 一级资本净额、客户存款规模 | 巴菲特 1995 致股东信 |
| **低成本负债**(Float) | 能否用低成本获取资金? | 净息差、负债成本率、保险浮存金规模 | 巴菲特 1995 致股东信:"Float 是我们最重要的资产" |
| **客户基础** | 客户数量 + 黏性? | 客户数(零售/对公)、MAU、流失率 | 银行业经典 |
| **风控能力** | 不良率是否低于行业? | 不良率、拨备覆盖率、ROE 稳定性 | 信用风险经典 |
| **品牌信任** | 客户是否信任品牌? | 品牌搜索量、客户主动选择率 | Kotler《Marketing Management》 |
| **科技/数据能力**(现代) | 数字化是否构成壁垒? | 数字客户占比、AI 风控准确率、专利数 | 招商银行案例 |

### 反问机制(金融业)

通用三轮反问见 §8。**金融业侧重点**:
- 可复制性:牌照壁垒多久才能拿到?资本金多久能积累?
- 周期韧性:经济下行时不良率上升多少?(优秀 < 50bps,警戒 > 100bps)
- 隐蔽威胁:利率市场化压低净息差、互联网平台分流、金融科技颠覆

## 8. 护城河三轮反问(统一机制,所有框架适用)

> **方法论背书**:这个反问机制融合了 Buffett 1995 致股东信"城堡 + 护城河"比喻、
> Howard Marks《Mastering the Market Cycle》(2018)"周期视角"、
> Helmer《7 Powers》"Power 持久性"概念。

### 第一轮:可复制性(2-5 年)

> 竞争对手要复制这个优势,需要多少资金?多少时间?目前有没有竞对正在复制?

- 2 年内可复制 → 降级为**窄护城河**
- 5 年内可复制 → 维持评级
- 5+ 年难复制 → 维持**宽护城河**

### 第二轮:周期韧性

> 如果行业需求下降 20%,这个护城河还能保护公司的利润率吗?

- **历史下行期测试**:用三表数据验证公司历史下行期表现
- 例:2018 年贸易战、2020 年疫情、2022 年加息周期
- 毛利率压缩 < 3pct → 护城河强
- 毛利率压缩 3-10pct → 护城河中
- 毛利率压缩 > 10pct → 护城河弱

### 第三轮:隐蔽威胁(短期 3 年 + 长期 5-10 年)

> 有没有当前没注意到的新技术/新模式/新政策正在侵蚀这个护城河?

- **3 年内威胁**:新进入者、产能扩张、技术替代
- **5-10 年威胁**:监管变化、技术革命、消费习惯迁移
- `westock-data news search <公司名> 行业 变革 颠覆 政策`（附当前年份）

### 反问记录表(每轮)

每轮记录:**反问问题 → 分析过程 → 评级是否调整 → 理由**。

## 9. 最终护城河综合评级(5 档)

| 评级 | 含义 | 典型代表 |
|------|------|---------|
| **极宽** | 多重 Power 复合,5+ 年难撼动 | 茅台、苹果、台积电、微信 |
| **宽** | 单一 Power 但极强,3-5 年难撼动 | 苹果(品牌+生态)、宁德时代(规模+客户认证) |
| **中-宽** | 单一 Power 较强 + 辅助 Power 1-2 个 | {制造业细分龙头}(技术+规模)、{医疗器械龙头}(产品+销售) |
| **中** | 1 个 Power 但被部分削弱 | 多数中型制造/消费 |
| **窄** | 0-1 个 Power,且持续被削弱 | 多数传统零售、传统媒体 |
| **无** | 没有可持续 Power,纯周期/纯资源 | 多数强周期股、纯 OEM |

### 综合评级输出模板

```
**最终护城河综合评级**:{极宽 / 宽 / 中-宽 / 中 / 窄 / 无}
**护城河类型**:{品牌型 / 规模型 / 网络效应型 / 转换成本型 / 牌照型 / 数据型 / 工艺型}
**核心逻辑**:{一句话说明护城河的主要来源}
**最大侵蚀力量**:{未来 3-5 年最可能侵蚀护城河的因素}
**支撑依据**:{2-3 个关键数据点 + 来源}
```

## 11. 竞争格局收尾:「为啥非它不可」3 问(Step3 §3.4 使用)

> **位置**:在 §3.1 识别竞对 + §3.2 业务对比之后,§3.3 护城河分析之前——从客户视角做竞争格局收尾,作为护城河分析的过渡。
> **3 问骨架通用**(Hamilton Helmer 7 Powers 核心三件套),**具体指标按行业适配**——下表是 6 套护城河框架对应的 Q1/Q2/Q3 指标定义。
> **与 §8 通用反问的区分**:§8 问的是"竞对能不能复制你的护城河";"为啥非它不可"问的是"客户为什么非选你不可"——两个视角互补。

### 11.1 3 问骨架(通用)

- **Q1 不可替代性** — 客户为什么选这家公司而非竞品
- **Q2 议价稳定性** — 客户切换会失去什么定价/费率优势
- **Q3 客户锁定** — 客户切换的成本(数据/认证/关系/习惯)

### 11.2 各护城河框架的 Q1/Q2/Q3 指标定义

> 按 §3.3 选定的护城河框架(业务模式标签 → 框架 A-F)查对应行。

| 框架 | 业务模式 | Q1 不可替代性 | Q2 议价稳定性 | Q3 客户锁定 |
|------|---------|--------------|--------------|------------|
| **A 消费品牌** | 消费品/零售 | 品牌心智 / 渠道卡位 / 品类占位(如茅台=高端白酒) | **提价权**——品牌溢价、单价稳定/提升证据 | **复购率/会员留存**——5+ 年品牌沉淀 |
| **B 平台/网络效应** | 互联网/平台/电商/社交/内容/游戏/SaaS | 跨边网络效应 / 双边规模 / 数据飞轮 | **单价/RPM 稳定**——抽佣率稳定、RPM 不降证据 | **DAU/MAU/留存**——使用时长、转换成本 |
| **C 制造业** | 制造/重资产/材料/能源/汽车零部件 | 工艺 know-how / 客户认证 / 规模效应 | **合同价稳定性**——订单单价年降幅度 ≤ 通胀、原材料顺价能力 | **认证锁定**——客户认证周期(汽车 2-3 年/医疗 3-5 年)+ 续约率 |
| **D 金融业** | 金融/银行/保险/券商/资管 | 牌照 / 资本金 / 渠道 | **净息差/费率稳定**——净息差稳定或走阔、综合费率证据 | **AUM 留存**——AUM 增长率、客户数、续保率 |
| **E 医药** | 医药/生物科技/医疗器械 | 化合物专利 / 管线深度 / 销售网络 | **专利期议价权**——创新药专利期定价权、集采压力下的差异化 | **处方/医生黏性**——医生处方习惯、医院覆盖率 |
| **F 地产/建筑** | 地产/建筑 | 拿地能力 / 融资成本 / 土储质量 | **房价/客单价议价权**——产品溢价、去化率 | **客单价/老带新**——复购率(写字楼租赁续约)、老带新比例 |

### 11.3 输出模板

```
**3 问验证**:

**护城河初判**:{极强/强/中/弱}(取决于 3 问回答的强度)
```

### 11.4 质量门

- [ ] Q1-Q3 每问有明确回答(不是"待评估"或"不清楚")
- [ ] 1 句定论与 3 问回答方向一致
- [ ] 护城河初判有数据支撑(不空喊"宽护城河"却没证据)
- [ ] Q1/Q2/Q3 指标已对照 §11.2 选定的护城河框架行(对号入座)
